队列

队列

概念

队列也是一种线性结构。 相比数组，队列对应的操作是数组的子集。 只能从队尾添加元素，只能从队首取出元素。是一种先进先出（FIFO）的数据结构。

应用

撤销操作——编辑器 系统调用栈——操作系统 括号匹配——编译器

方法

  • void enqueue(E e) ：队尾添加一个元素e
  • E dequeue() ：取出队首元素
  • E getFront() ：查看队尾元素
  • int getSize() ：返回队列长度
  • boolean isEmpty() ：返回是否空队列
